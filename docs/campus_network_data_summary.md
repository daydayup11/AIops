---

## 一、数据库概览

共 1 个数据库 `logdb`，包含约 60 张核心表（另有同名 `_rep` 副本表）。  
核心数据分为以下几类：

| 类别 | 主要表 |
|------|--------|
| 会话/流量 | `sessions`, `sessions_ipv6`, `natlog`, `npm` |
| 应用层日志 | `httplog`, `url`, `dns`, `pdns`, `applog` |
| 安全/事件 | `event`, `ids`, `alert`, `sevt` |
| 认证/用户 | `usrauth`, `auth_account`, `dhcp`, `cgnat` |
| 统计汇总 | `flow_statis`, `if_traffic`, `device_traffic`, `iplog`, `wanlog` |
| 资产管理 | `assets4`, `assets6`, `myassets` |
| 系统日志 | `syslog`, `operation_log` |

## 二、核心表字段说明

### `sessions` / `sessions_ipv6` — 网络会话记录（最核心）

记录每条 TCP/UDP/ICMP 会话的完整五元组及流量信息。

| 字段 | 类型 | 说明 |
|------|------|------|
| `dev_id` / `probe_id` | UInt16 | 设备/探针 ID |
| `start` / `end` | DateTime | 会话开始/结束时间 |
| `protocol` | UInt8 | 协议号（6=TCP, 17=UDP, 1=ICMP）|
| `src_ipv4` / `dst_ipv4` | IPv4 | 源/目标 IP |
| `src_port` / `dst_port` | UInt16 | 源/目标端口 |
| `src_nat_ipv4` / `src_nat_port` | IPv4/UInt16 | NAT 后源地址/端口 |
| `up_bytes` / `down_bytes` | UInt64 | 上行/下行字节数 |
| `up_pkts` / `down_pkts` | UInt32 | 上行/下行包数 |
| `ret_up_pkts` / `ret_down_pkts` | UInt32 | 重传包数 |
| `ifname` / `out_ifname` | String | 入/出接口名 |
| `wan_name` | String | WAN 线路名称 |
| `appid` | UInt32 | 应用识别 ID |
| `domain_name` | String | 关联域名 |
| `account` | String | 用户账号 |
| `src_ISP` / `dst_ISP` | String | 源/目标 ISP |
| `src_pos_*` / `dst_pos_*` | String | 源/目标地理位置（国/省/市/区）|
| `tcp_flags` | UInt16 | TCP 标志位 |
| `direction` | UInt8 | 流方向 |
| `dscp` / `tos` / `ecn` | UInt8 | QoS 标记 |
| `malc_hit` | UInt8 | 恶意流量命中标记 |
| `goodflow` | UInt8 | 优质流量标记 |
| `priod_min` / `up_avg_min` / `down_avg_min` | Float32/UInt32 | 分钟级速率统计 |

`sessions_ipv6` 结构相同，IP 字段替换为 IPv6，额外有 `flow_label`（IPv6 流标签）。

## 五、可用于研究的分析方向

1. **用户行为分析**：结合 `sessions` + `usrauth` + `auth_account`，可关联用户身份与流量行为
2. **应用识别与分类**：`appid` 字段贯穿多张表，可分析各类应用的流量占比与趋势
3. **出口链路质量**：`npm` 表含延迟/重传数据，`wanlog` 含各线路带宽，可做多链路质量对比
4. **DNS 行为分析**：大量 DoH（dns.google、doh.pub）查询，可研究 DNS 加密使用情况
5. **异常流量检测**：`event` 表记录 QQ/微信登录，`malc_hit` 字段标记恶意流量，`ids` 表备用
6. **地理流量分布**：目标国家/ISP 字段可分析境外访问模式
7. **终端设备分析**：`src_mac` 可关联设备类型，`dhcp` 表（待启用）可补充设备信息
